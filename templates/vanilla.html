AsterMind Chatbot Demo

This page demonstrates the AsterMind Chatbot widget using vanilla JavaScript. Click the chat bubble in the bottom-right corner to open the chat.

About This Demo

The chatbot widget is loaded using a single script tag and initialized with JavaScript. No build tools or frameworks required.

Try asking questions about your knowledge base. The chatbot will stream responses in real-time and show source citations when available.

Configuration Used

AsterMindChatbot.init({
  apiUrl: 'https://your-api-url.com',
  apiKey: 'your-api-key',
  position: 'bottom-right',
  theme: {
    primaryColor: '#4F46E5',
    borderRadius: '16px'
  },
  greeting: 'Hi! How can I help you today?',
  placeholder: 'Ask me anything...',
  headerTitle: 'AsterMind Assistant',
  headerSubtitle: 'Powered by AI',
  agent: {
    enabled: true,
    siteMap: [
      { path: '/products', name: 'Products' },
      { path: '/contact', name: 'Contact' }
    ]
  },
  fallback: {
    enabled: true,
    message: 'Working in offline mode'
  },
  onReady: function() {
    console.log('Chatbot ready!');
  },
  onMessage: function(message) {
    console.log('Message:', message);
  }
});
